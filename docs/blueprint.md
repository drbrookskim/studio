# **App Name**: Planning Lens

## Core Features:

- Rich Text Editor: Enable authors to create and format articles with a rich text editor, supporting various media types, styles, and layouts, as seen in https://brunch.co.kr/@drbrooks.
- Article Publishing: Allow authors to publish articles with a title, content, and metadata, with draft saving capability.
- Category Management: Implement a system for categorizing posts, so they may appear in multiple lists or filters for easy navigation.
- Focused Article Reading View: Provide a distraction-free reading view that highlights the article content, mirroring the clean design of https://drbrooks.tistory.com/ and Brunch.
- AI Article Summary: Generate a concise summary of each article, leveraging AI to capture the main points, that readers may then review for relevance, or allow them to gauge its complexity as a tool for informing whether they will have the available planning to digest the whole thing.
- Full-Text Search: Enable users to search for articles based on keywords found within the content.

## Style Guidelines:

- Primary color: Deep blue (#2C3E50) to evoke a sense of calm and focus.
- Background color: Off-white (#F5F5F5) to provide a clean, readable backdrop.
- Accent color: Soft gray (#95A5A6) for secondary text, borders, and subtle UI elements.
- Headline font: 'Playfair' serif for titles, lending an elegant, fashionable, high-end feel; body text: 'PT Sans' sans-serif for body copy; for enhanced readability, especially on longer articles.
- Use simple, outlined icons for categories and navigation elements, maintaining a minimalistic design.
- Emulate Brunch's clean layout, using ample whitespace and a focus on readability and article presentation.
- Incorporate subtle transitions and animations for a smooth user experience when navigating between articles and sections.
